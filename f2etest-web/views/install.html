安装快捷方式到Chrome

  1. 打开：Chrome官方扩展市场并安装扩展 https://chrome.google.com/webstore/detail/f2etest-client/nbfokjfplbppkihmcocbphbkgbfnadjd
  2. 安装好后，点击Chrome右上角图标
  3. 初次使用，会提示初始化，在打开的选项界面中，输入：https://<%=host%>，并点击保存按钮
  4. 打开需要测试的站点，例如：http://www.alibaba.com/，随后，再次点击Chrome右上角图标
  5. 鼠标点击需要测试的浏览器，即会在目标浏览器中打开当前网页的URL(按住Ctrl可同时打开多个浏览器)

下面将F2etest所有应用的快捷方式安装至您的桌面，此种快捷方式由于是直连，因此速度上可以得到极大提升，获得几乎接近本地应用的用户体验，强烈建议您使用此方式。

安装快捷方式到Windows桌面 (Win7)

  1. 打开：控制面板，打开：RemoteApp 和桌面连接
  2. 打开：使用RemoteApp和桌面连接设置一个新连接
  3. 输入：https://<%=host%>/rdfeed-<%=apiKey%>.xml
  4. 点击两次：下一步，再点击：完成
  5. 下载脚本：install.bat，下载后双击执行
  6. 打开：开始菜单，打开：所有程序
  7. 打开：RemoteApp 和桌面连接
  8. 在f2etest文件夹上右键，选择：发送到，再选择：桌面快捷方式
  9. 双击打开桌面上新建的快捷方式：f2etest
  10. 打开需要的浏览器，即可开始测试之旅

安装快捷方式到Mac桌面

  1. 请先安装：Microsoft Remote Desktop。由于此软件只存在在英文市场，因此必需使用Apple英文市场的ID进行安装。（公共英文账号: f2e.sc.icbu@gmail.com 密码: helloF2E ）
  2. 打开刚才安装好的：Microsoft Remote Desktop程序
  3. 点击：Remote Resources按钮
  4. URL处输入：https://<%=host%>/rdfeed-<%=apiKey%>.xml
  5. User name处输入用户名: <%=userid%>
  6. Password处输入密码: <%=remotePassword%>
  7. 点击：Refresh按钮，此时正常情况下，应该在界面上可以看到F2etest上所有浏览器
  8. 打开需要的浏览器，即可开始测试之旅
